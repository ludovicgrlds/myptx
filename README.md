# MyPTX

Application de gestion des véhicules pour centre de secours.

## Stack
- React + Vite
- Supabase (auth + base de données)
- Déployé sur Vercel

## Installation locale
```bash
npm install
npm run dev
```

## Déploiement
1. Pousser sur GitHub
2. Connecter le repo sur Vercel
3. Ajouter les variables d'environnement dans Vercel :
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
